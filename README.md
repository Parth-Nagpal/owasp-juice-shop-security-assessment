# OWASP Juice Shop Vulnerability Assessment

## Overview
This project documents a hands-on web application security assessment performed on OWASP Juice Shop, a deliberately vulnerable web application. The assessment focuses on identifying, validating, and documenting common web vulnerabilities using manual testing techniques.

## Scope
- Target Application: OWASP Juice Shop
- Testing Type: Black-box testing
- Environment: Local/Lab setup

## Tools Used
- Burp Suite Community Edition
- Web Browser (Chrome/Firefox)

## Methodology
- Manual request/response inspection
- Input validation testing
- Vulnerability verification through proof-of-concept payloads
- Impact analysis and remediation recommendations

## Key Findings
- Stored Cross-Site Scripting (XSS)
- Input validation weaknesses

## Learning Outcomes
- Practical understanding of OWASP Top 10 vulnerabilities
- Hands-on experience with Burp Suite
- Writing structured security assessment reports

## Disclaimer
This project was conducted in a controlled lab environment for educational purposes only.
